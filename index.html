How this works

Every time you click 'Check In & Refill' on a map marker, we assume you saved one single-use plastic bottle.

  • **1 Bottle saved** ≈ 5g of plastic avoided.
  • **1 Bottle saved** ≈ $2.00 in cost savings.
  • **1 Bottle saved** ≈ 0.2kg of CO2 emissions offset.

About Bubblr

Bubblr is a progressive web application (PWA) designed to help you find public water refill stations (bubblers) and track your personal environmental impact from choosing reusables over single-use plastic.

Our Data Sources

We combine two layers of data to provide comprehensive coverage:

  • **Internal CSV Data:** Curated, filtered, and verified locations (e.g., from local government sheets).
  • **Community Data:** Real-time data fetched from the **OpenStreetMap/Overpass API** for global, community-contributed locations. This layer is dynamically loaded as you move the map.

Technical Stack

This application is built entirely on modern web technologies:

  • **Frontend:** HTML, JavaScript, Leaflet.js (for maps), and Tailwind CSS.
  • **Backend:** Serverless **Netlify Functions** written in JavaScript to securely proxy requests to external APIs (OpenStreetMap and Google Sheets), avoiding CORS issues and API key exposure.
